**Special data:** create a new folder with your user name. Use sufficient filenames, to understand the meaning. You can create a help text like this, when you upload a file called "index.html" to your directory. The file may contain html, but a pure textfile (eg. written with notepad) is sufficient. Please consider the following options for upload: * Your data is for one dataset (and therefore site, user and instrument). Use this [Excel template](/datafiles/dataset-xxxx.xls"). Prior to upload, you should create a fitting dataset with all relevant meta-data. Make sure that "dataset-" is part of the filename. * Your data is collected at serveral (existing) sites and should append existing datasets. Use the template for routine measurements. Make sure your filename ends with log.xls * Your data is collected at several sites, that are distributed over the study area. There is little or no chance data will collected at exactly this site again. At the moment, there is no way to include this type of data in the database. Just do an upload to the server and write a small description of your data in an index.html file, and upload it also.
