This is the root folder. To upload data please use the correct folder:
  • Logger data: select the folder for the logger and follow the descriptions there
  • Routine manual measurements: goto the folder manual measurements. An Excel template to collect the data can be found there.
  • Special data: create a new folder with your user name in the "user" directory.
  • Manuals and handbooks: Manuals and handbooks can be found in the folder manuals. Please uplaod relevant manuals to that directory. You can write a link to a manual to the instruments page.
