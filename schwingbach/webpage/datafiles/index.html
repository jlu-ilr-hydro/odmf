This is the root folder. To upload data please use the correct folder: - __Logger data:__ select the folder for the logger and follow the descriptions there - __Routine manual measurements:__ goto the folder dir:manual-measurements . An Excel template to collect the data can be found there. - __Special data:__ create a new folder with your user name in the "user" directory. - __Manuals and handbooks:__ Manuals and handbooks can be found in the folder manuals. Please uplaod relevant manuals to that directory. You can write a link to a manual to the instruments page.
