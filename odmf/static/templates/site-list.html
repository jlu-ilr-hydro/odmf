Import sites

Importing sites is not implemented yet. You can upload tabular data (csv, excel, parquet) or geojson files into the filemanager and ask a site administrator to create sites using the programming interface. Note the hints below, though.

Import tabular data

When you import sites, note that the tabular data must have the columns name, lat, long. Optionally the column names height, icon, comment are used.

Import geojson data

If you are importing geospatial data in geojson form, the projection of the data must be geographical with WGS84 ellipsoid. Each feature must have a name property. Optionally property names height, icon, comment are used. GeoJSON files can be created with QGIS from most geospatial data, tutorials are simple to find with common search machines, like this one: How_to_create_a_GeoJSON_in_QGIS
